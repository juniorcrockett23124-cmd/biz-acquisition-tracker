# Job 5: Weekly Shortlist

## Goal

Produce a concise top-priority list for fast review.

## Rules

1. Read `data/candidates.csv`.
2. Rank best current opportunities.
3. Create `data/weekly-shortlist/latest.md`.
4. Include:
   - top 5 overall
   - top 3 ready for outreach
   - top 3 ready for underwriting
   - top 3 pass candidates and why
